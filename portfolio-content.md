### Project 3 — Academic Help Assistant

**Type:** Practice Project
**Status:** Completed (API-dependent features ready for client deployment)

**Description:**
Built a platform that connects students with subject experts who guide them through assignments and research work. Students can reach out to relevant experts, and experts can provide structured academic guidance.

Beyond the core connection feature, I developed three additional tools:
- **Plagiarism Detector** — checks submitted content for originality
- **AI Content Detector** — identifies AI-generated text in academic submissions
- **Turnitin Integration** — connects with Turnitin's database for similarity checking

The plagiarism, AI detection, and Turnitin features require paid API keys tied to their main databases. I built and tested the full integration logic — the features are production-ready. API keys were not purchased since this was a practice project, but can be activated for any client who needs them.

**Tech Used:** React, Node.js, PostgreSQL, REST APIs
**Skills Demonstrated:** Full-stack development, third-party API integration, academic tools, student-expert matching systems

## Notes for Development

- **Tone:** Direct, confident, human — not corporate, not AI-sounding
- **Design Direction:** Clean, minimal, modern — dark or light mode (decide before build)
- **Target audience:** School owners, academy principals, online course creators in Pakistan (and globally)
- **No fluff sections** like "My Values" or generic skill bars with percentages
- **Avoid:** Stock illustrations, excessive animations, hero images with code screenshots
- **Consider:** One strong hero statement, project case-study style cards, simple contact form or WhatsApp CTA
